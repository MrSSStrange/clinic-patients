# Реестр пациентов

Учебный проект без фреймворков: список пациентов, быстрый поиск и карточка с основной информацией о приёме.

**Демо:** https://mrssstrange.github.io/clinic-patients/

## Возможности

- поиск по имени, диагнозу и врачу;
- фильтр по отделению и оплате;
- сортировка по дате приёма, возрасту и риску;
- добавление и удаление записей;
- загрузка собственного фото;
- сохранение изменений в `localStorage`.

## Как открыть

Достаточно открыть `index.html` в браузере. Сборка и установка зависимостей не нужны.

Все имена и медицинские данные в проекте вымышлены.
